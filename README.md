# Bumblebee Gut Microbiome Oxygen Tolerance Analysis
Metagenomic analysis of bumblebee gut microbiome. The general goal of the project will be to find a relationship that explains the differences between bumblebees with different levels of oxygen tolerance. Samples across 20 different bumblebee species in the US and Colombia are used for analysis. The bioinformatic tool OxyMetaG is utilized to predict the aerobe/anaerobe of the bumblees sampled using 20 pfam identified
